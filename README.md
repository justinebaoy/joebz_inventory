# joebz_inventory

## System Overview

**joebz_inventory** is a web-based **Quantitative Research system** focused on inventory and stock data management.

### What the system is
- A PHP + MySQL inventory platform for collecting, organizing, and tracking item data.
- A system intended to support **quantitative research workflows** by keeping records structured and measurable.


### What it is used for
- Recording and maintaining inventory information.
- Monitoring stock quantities and item updates over time.
- Providing organized data that can be used for analysis, reporting, and research decisions.


## Run on localhost

### Requirements
- PHP (8.x recommended)
- MySQL / MariaDB
- Web browser


### Steps
1. Clone the repository and open it locally.
2. Create a MySQL database (example: `inventory_joebz`).
3. Update database connection settings in `config/db.php` if needed.
4. Start a local PHP server from the project root:


```bash
php -S localhost:8000

}
```

5. Open the app in your browser:
```text
http://localhost:8000
```
